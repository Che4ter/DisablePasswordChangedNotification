Disable the password changed notification introduced in Wordpress 4.4

== Description ==
Disable the password changed notification introduced in Wordpress 4.4


== Installation ==
From your WordPress dashboard
    Visit \'Plugins > Add New\'
    Search for \'Disable Password changed notification\'
    Activate the Plugin

Manual Installation (From WordPress.org or Focus.ch)
    Download \'PWCN.zip\'.
    Upload the \'PWCN\' directory to your \'/wp-content/plugins/\' directory, using your favorite method (ftp, sftp, scp, etc...)
    Activate the Plugin

The Plugin has no settings
